# trojan-go-panel
trojan-go 多用户多节点管理面板

## 进度😋
基础功能已经完成，有一定动手能力的小伙伴们可以部署了！
如果还要等完全体，建议下周末（2021-07-11）再来看看！

## 部署

### 主节点部署

`source <(curl -sL https://git.io/install-trojan)`

### 子节点部署
`source <(curl -sL https://git.io/install-trojan) --worknode 子节点域名 主节点域名 子节点名称 主节点数据库密码`
